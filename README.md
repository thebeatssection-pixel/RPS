## Installatie & Gebruik

1. **Installeer dependencies:**
   ```bash
   npm install
   ```

2. **Start de server:**
   ```bash
   npm start
   ```

3. **Open het spel:**
   Ga naar `http://localhost:3000`

## Deployment

Dit project gebruikt een Node.js backend. Je kunt het **niet** hosten op GitHub Pages (dat is alleen voor statische sites).

Om dit online te zetten (zodat vrienden op andere computers kunnen spelen), raad ik **Render** of **Railway** aan:

1. Push deze code naar GitHub.
2. Maak een account op [Render.com](https://render.com).
3. Kies "New Web Service" en connect je GitHub repo.
4. Render snapt automatisch dat het een Node app is (`npm start`).
5. Je krijgt een unieke URL om te delen!
